### Installation

- Golang: 1.21
- Get bot [token](https://helpdesk.bitrix24.com/open/17622486/), [chat id](https://stackoverflow.com/questions/32423837/telegram-bot-how-to-get-a-group-chat-id)
- Run:
  - ```golang
    cp env.example .env
    ```  
  - ```golang
    go run cmd/main.go
    or 
    make run
    ```
